# OSPP Name Usage Guidelines

"OSPP" and "Open Self-Service Point Protocol" are not registered trademarks. They are used to identify this protocol specification and related projects.

## Usage Guidelines

To avoid confusion, please follow these guidelines when referencing OSPP:

- **Do** use "OSPP" or "Open Self-Service Point Protocol" to refer to this specification.
- **Do** use "OSPP-compatible" or "OSPP-compliant" to describe implementations that pass the conformance test suite.
- **Do not** use "OSPP" in a way that implies official endorsement unless your implementation has been validated against the conformance suite.
- **Do not** use "OSPP" as part of a product name in a way that could confuse users into thinking the product is the protocol itself.

## Conformance Claims

Implementations **SHOULD** reference the specific version they target:

> "This product implements OSPP v1.0.0-draft.1 (Core + Transaction profiles)."

Avoid unqualified claims like "OSPP Certified" — there is no formal certification program at this time.

## Questions

If you have questions about using the OSPP name, open an [Issue](https://github.com/ospp-org/ospp/issues).
